Add custom lightbox attribute to linked flash files or images in posts, pages and comments, group images using ID.

== Description ==

The Lightbox plugin allows you to add custom attribute (choosing rel or class) to linked images in 
posts, pages and comments.Image formats supported are bmp, jpg, gif, jpeg and png. You can use this plugin with other advanced 
lightbox plugins.

Is not include Javascript files for any lightbox plugins, you must include yourself via your theme.

Additional very simply you can group your images for each post by adding [id] in your rel or class and you can also add a custom attribute to 
linked Flash files (swf and flv).

== Installation ==

1. Unzip archive and upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Adv New Lightbox' in the settings menu
4. Add custom attributes for flash files images and flash files (optional) and click save
5. For grouping use [id] for example `rel="lightbox[id]"`
